Feature Ideation

Possible UI Solutions

Safety and Comfort

  • Filter by bike types to narrow down search results.
  • Safety Rating: Display safety ratings prominently for each bike. Possible through a scale range (lower safety ratings describe better fit for professionals and tricks, whereas higher safety ratings are better for beginnings/older populations riding casually).

Beginner Friendly

  • Popup with a brief description next to each bike specification (i icon) briefly outlining what each quality means and why it's important.
  • "For Beginners" title in the navigation bar linked to Beginner's Guide blog. Provides an overview all necessary considerations for new riders.

Aesthetics

  • Preview button for bike cards in home page for clear images of bikes from different angles - clarify what the bike will look like for anxious users/users who care about looking cool.
  • Suggested/recommended accessories section for each selected bike: "This would go well with..." for users who want to stand out/decorate their bikes.
  • User Reviews: Allow users to upload images and reviews of their customized bikes. Users also should be able to filter the reviews (With Images Only) and only acess reviews with images.
  • Color Customization: Allow users to visualize how different color options would look on the bike.

Affordability

  • Filter bikes based on budget range, ensuring filters act as filters, not sort options. Users can click on "budget" input their range, which causes the system to only return bikes that fit within that range.

Intent

  • Clear category/terrain type tag on the top of each bike item page.
  • Tag indicating "Best for XYZ" based on user personas' scoring.

Maintenance

  • Description of estimated lifespan for each bike.
  • Maintenance Tips for each bike model.

Customization

  • Pop-up indicating suitable heights for each bike.
  • Sort by height button with an equation for users to input their height and get relevant recommendations through a scale. The scale will return where their their measurements position within the desired range for the product.

Smart Consumerism

  • "Latest Technology" navigation category for users to find cutting-edge features.
