Evaluation Criteria: Under the Grouping Tab, we've color coded each grouping, and assigned it to each user persona. In the scoring tab, we used COUNTIF functions to tally up how much each grouping was important, deemed by the total of x's placed under each category. Lastly we used a combination of INDEX, MATCH, MAX, and COUNTIF to discover which user persona appeared to care the most about each category.
